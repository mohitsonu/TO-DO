# TO-DO Application

This is a simple TO-DO application built with React. It allows users to add, complete, and delete tasks. The application also displays a daily motivational message.

## Deployment

The application is deployed on GitHub Pages. You can access it using the following link:

[TO-DO Application](https://mohitsonu.github.io/TO-DO)

## Features

- Add new tasks
- Mark tasks as complete
- Delete tasks
- Display daily motivational message

## Design

The application is designed using React and CSS. The main components of the application are:

- `App.js`: The main component that holds the state and renders other components.
- `Header.js`: The header component.
- `TaskList.js`: The component that displays the list of tasks.
- `AddTask.js`: The component that allows users to add new tasks.
- `Achievements.js`: The component that displays achievements.
- `Motivation.js`: The component that displays the daily motivational message.

### CSS

The application uses CSS for styling. The main styles are defined in the `App.css` file. The styles ensure that the application has a clean and modern look.
